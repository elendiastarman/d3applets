Tametsi Puzzles

I love the game Tametsi so much that I even made puzzles for it.
These puzzle files should be placed within the game's Steam data, e.g. within the folder D:\SteamLibrary\steamapps\common\Tametsi\puzzles.

A combination lock puzzle is one where all tiles are covered at the start, there is only one color, and the only way to make progress is to use the column hints and the hints revealed when clearing tiles.
The "score" for combination lock puzzles is a rough measure of its difficulty that attempts to measure not only the difficulty of individual logic steps but also the consistency between subsequent logic steps. For example, two consecutive logic steps that are both medium difficulty scores higher than an easy step followed by a hard step or vice versa.

There is a subreddit r/Tametsi and the code that I used can be found at my GitHub repository Tametsi-level-generator.
